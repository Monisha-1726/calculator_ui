# 🧮 Calculator App

A simple web-based Calculator application built using **HTML, CSS, and JavaScript**.
This application allows users to perform basic arithmetic operations through a clean and responsive user interface.

---

## 🚀 Features

* Perform basic calculations (Addition, Subtraction, Multiplication, Division)
* Clear button to reset the display
* Simple and user-friendly interface
* Responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

```
calculator-app/

```

## ⚙️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the **index.html** file in your browser.
